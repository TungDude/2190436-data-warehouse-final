## Current Raw Ingestion Behavior

- [x] Historical data is stored as monthly CSV files named
  `chicago_crime_<start>_to_<end>_<timestamp>.csv`.
- [x] Future scheduled data is stored as one CSV file per source day named
  `chicago_crime_<date>_<timestamp>.csv`.
- [x] Raw files are append-only and partitioned by `ingest_date`.
- [x] The daily Lambda fetches `current Chicago date - 8 days` because the
  Chicago source excludes the most recent seven days.
